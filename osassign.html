1. What is Linux?

Linux is a free and open-source operating system used on servers, desktops, and mobile devices. It is known for its stability, security, and flexibility.

2. Basic Linux Tasks

Task Command Purpose
See current directory pwd Displays current folder location
List files ls Shows files and directories
Change directory cd foldername Move into a different folder
Create a file touch filename.txt Creates a new empty file
Create a folder mkdir foldername Makes a new folder
Delete a file rm filename.txt Deletes a file
Delete a folder rmdir foldername Removes an empty folder
Edit a file nano filename.txt Opens file in terminal editor
See file content cat filename.txt Displays file content
Copy a file cp old.txt new.txt Duplicates a file
Move/Rename file mv old.txt new.txt Moves or renames file
Install software sudo apt install packagename Installs new software
Update system sudo apt update && sudo apt upgrade Updates Linux system
